
1) В данный момент реализован алгоритм копирования чертежей компаса в папке D_bag на 4 папки внутри неё с сохранением каждого чертежа с литерами Э, О, О-О1 и без литеры и к каждому сохраняеться PDF


при запуске update.bat - к каждому файлу cdw и swp применяеться алгоритм из файла algoritm.py   
при необходимости алгоритм переделываеться согласно возможностям модуля KOMPAS-tools: https://pypi.org/project/KOMPAS-tools/
для работы необходимо указать внутри путь к интропритатору питона

Модуль KOMPAS-tools расширен на функции смены литер 
            kompas.set_litera_1("Литера")
            kompas.set_litera_2("Литера")
            kompas.set_litera_3("Литера")
    на функции сохранения, открытия и удаления файла из окна компаса
            kompas.open_document("Путь к файлу открытия")
            kompas.save_document("Путь к файлу сохранения")
            kompas.close_document(True|False(сохранять|нет))

Сборщик000.bat - утилита для виндоус, собирает из папки в которой находиться все файлы с расширением *.cdw и *.spw и закидывает их в папку \hom

Утилиты get_CDW.bat, get_SWP.bat, get_PDF.bat - перемещают файлы с соответсвующим расширениям из всех своих поддеректорий и дериктории в папки D_CDW, D_SWP, D_PDF
